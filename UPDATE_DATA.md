# 小白更新数据步骤

## 更新病例数据

1. 打开 GitHub 仓库。
2. 进入 `data` 文件夹。
3. 点击 `timeseries.json`。
4. 点击右上角铅笔图标。
5. 在最后一个 `}` 后面加英文逗号，再粘贴一条新数据。
6. 确认最后一条数据后面没有逗号。
7. 点击 `Commit changes`。
8. 等待 Netlify 自动部署完成。

## JSON 最容易出错的地方

- 逗号必须是英文逗号 `,`
- 双引号必须是英文双引号 `"`
- 最后一条数据后面不要有逗号
- 日期建议保持 `YYYY-MM-DD`
- 空值用 `null`，不要写成中文“无”
