部署为真正动态网页

纯 HTML 不能稳定抓取 WHO、Reuters、AP 等所有网页，原因包括 CORS、反爬、付费墙和网页结构变化。建议：

  1. 前端使用本页面；
  2. 后端每 1–3 小时抓取 WHO/CDC/ECDC/RSS/GDELT；
  3. 用正则或 LLM 抽取“确诊/死亡/康复/地区/日期”；
  4. 人工审核后写入 JSON；
  5. 前端只读取审核后的 JSON，避免误报。
